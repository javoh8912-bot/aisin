# Avenaa AI

Avenaa AI - Bu aqlli chat va P2P xabar almashish platformasi.

## Xususiyatlari
- Gemini AI bilan chat qilish
- P2P (foydalanuvchilararo) xabarlar
- Global sozlamalar (API kalitni boshqarish)
- Toza va premium dizayn

## O'rnatish

1. Loyihani yuklab oling:
   ```bash
   git clone <repo-url>
   cd ai
   ```

2. Kerakli kutubxonalarni o'rnating:
   ```bash
   npm install
   ```

3. Serverni ishga tushiring:
   ```bash
   npm start
   ```

4. Brauzerda oching: `http://localhost:3000`

## Muhim
Loyiha ishga tushishi uchun sozlamalar bo'limidan **Gemini API Key** ni kiritishingiz kerak. Bu ma'lumotlar serverda `settings.json` faylida saqlanadi va `.gitignore` orqali GitHub'dan yashirilgan.
